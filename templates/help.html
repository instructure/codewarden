Help

This coding competition host accepts source code files and runs them in a restricted environment. You should submit source code that complies with the associated problem description. If your language is interpreted, we'll run the interpreter for you. If your language is compiled, we'll compile and then run it.

The restricted environment is pretty restricted, both with resources and with capabilities. Specifically be careful with memory usage, most system calls, and runtime.

The input data we suggest will be coming from STDIN, and you need to write your output to STDOUT. Take a minute to figure out how to do this in your language.

The test cases use Unix-style line endings, so make sure your newlines end with just the \n newline character.

If you have some question, please feel free to email $(contact_email) to ask.

The supported language versions are:

    $for language in languages:
  • $language["visible_name"] - $language["version"]

Specific language notes:

  • Java/Scala - please name your main class 'Main'.
  • PHP - STDIN can be found at php://stdin.
